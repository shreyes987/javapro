TRUTH TABLES



Remember a T (true) is the same as 1 (on) and a F (false) is the same as 0 (off).


AND GATE TRUTH TABLE
The AND gate's truth table requires
2 true inputs for a true output.
By combing P and Q, the
output will be a T or F.
OR GATE TRUTH TABLE
The OR gate's truth table requires
1 or more true inputs for a true output
NOT (INVERTER) TRUTH TABLE
The NOT'S truth table demonstrates
how an input is changed to its complement.
For example: a true input would
become a false output



Now that you have reviewed the truth table
for the logic gates (AND, OR and NOT), we
will show you how to write a logic
expression from a truth table.


For example: If you were given this truth table, what would be the resulting logic expression (??)?








1. Look at each line in the table for which the result is 1 (TRUE).





2. For each of those:
a. Find the P and Q values.
b. Build an AND statement with variables (P, Q).
c. If the variable is 1 then it is itself (eg. P, Q)
d. If the variable is 0, then it is its negation (eg. P', Q').









3. Use OR's ("+") to connect the statements from step 2.
