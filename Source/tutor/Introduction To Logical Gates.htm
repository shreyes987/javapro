INTRODUCTION TO LOGICAL GATES



The building blocks of a circuit are called gates. There are at least three types of gates:
an AND gate, OR gate, and NOT gate. A NOT gate is also called an INVERTER.
When these gates are properly connected, it creates a logical circuit.

The top and bottom input values determine the state of the input. For
example if an input is 1, then current flows through the input line. If
the input is 0 then there is no current through the input line.


To illustrate how a logic gate operates, the examples below refer to a light switch.
When a light switch is down, no current flows through the line, thus keeping the
light off. However if the light switch is flipped up, a current flows through the
line to turn the light on.




AND GATE EXAMPLES

In an AND gate, both inputs have to be 1 in order for current to flow giving an output of 1. In this AND gate both the top and bottom inputs are 1. Notice the light is on.

OR GATE EXAMPLES

In an OR gate, only 1 input must be a 1 in order for current to flow giving an output of 1. This OR gate has both the input and the output a 1, so it's output is a 1.




This OR gate has 1 as it's top input and 0 for its bottom input. The 1 input has current flowing through its input line, which sends a current to the output line. Since there is at least 1 input, current is flowing, giving an output of 1. Notice the light is turned on.

This OR gate has 0 as its top input and 1 for its bottom input. The 1 input has current flowing through its input line, which sends a current to the output line. Since there is at least 1 input, current is flowing, giving an output of 1. Notice the light is turned on.
