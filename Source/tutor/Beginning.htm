AND GATE EXAMPLES
For the AND gate's output to be a "1" (ON-CURRENT), both inputs must be a "1" (ON). The first example's inputs are both "1" (ON) so the output is "1" (ON-CURRENT); however, in the second example, the top input P is a "1" (ON) but the bottom input Q is a "0" (OFF) so the output is a "0" (OFF-NO CURRENT).
In the first example, notice the top input P is a "0" (OFF) but the bottom input Q is a "1" (ON) so the output is a "0" (OFF-NO CURRENT). For the second example, both inputs P and Q are "0" (OFF) so the output is "0" (OFF-NO CURRENT)
OR GATE EXAMPLES
For the OR gate's output to be a "1" (ON-CURRENT), at least one input must be a "1" (ON). These two examples have at least one input with a "1" (ON) so the outputs are "1" (ON-CURRENT).
In the first example, notice the top input P is a "0" (OFF) but the bottom input Q is a "1" (ON) so the output is a "1" (ON-CURRENT). For the second example, both inputs P and Q are "0" (OFF) so the output is "0" (OFF-NO CURRENT).
NOT (INVERTER) GATE EXAMPLES
For the NOT gate's output to be a "1" (ON-CURRENT), the input must be a "0" (OFF). Fof the output to be a "0" (OFF-CURRENT), the input must be a "1" (ON). The NOT gate gives you the complement input value.
Input for P is 1 (ON), input for Q is 1 (ON). So when you evaluate the OR gate it's output is 1 (ON). The NOT gate complements Q's input of 1 to give an output of 0 (OFF). The inputs for the AND gate are the results of (P OR Q) AND (NOT Q). So the final resulting output is 0 (OFF).
Input for P is 1 (ON), input for Q is 0 (OFF). So when you evaluate the OR gate it's output is 1 (ON). The NOT gate complements Q's input of 0 to give an output of 1 (ON). The inputs for the AND gate are the results of (P OR Q) AND (NOT Q). So the final resulting output is 1 (ON).
Input for P is 0 (OFF), input for Q is 1 (ON). So when you evaluate the OR gate it's output is 1 (ON). The NOT gate complements Q's input of 1 to give an output of 0 (OFF). The inputs for the AND gate are the results of (P OR Q) AND (NOT Q). So the final resulting output is 0 (OFF).
Input for P is 0 (OFF), input for Q is 0 (OFF). So when you evaluate the OR gate it's output is 0 (OFF). The NOT gate complements Q's input of 0 to give an output of 1 (ON). The inputs for the AND gate are the results of (P OR Q) AND (NOT Q). So the final resulting output is 0 (OFF).
